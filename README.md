# Лендинг сайт "Авто из США под ключ"

Этот одностраничный лендинг-сайт предназначен для продвижения услуги по покупке авто из США "под ключ". Он выполнен с использованием современных технологий HTML, CSS и JavaScript, обеспечивая привлекательный внешний вид и интерактивность. Сайт включает яркую анимацию и адаптирован для комфортного просмотра на любых устройствах — от настольных компьютеров до мобильных телефонов. Цель проекта — привлечь клиентов и информировать их о преимуществах услуги.

## Используемые технологии

- HTML — создание структуры страниц
- CSS — стилизация и оформление элементов
- JavaScript — реализация анимаций и интерактивных функций
- Slick Slider - реализация слайдера
- WOW.js - анимации при скролле
  
## Ключевые особенности

- Адаптивный дизайн, автоматически подстраивающийся под размер экрана устройства
- Интерактивные анимации, делающие сайт более привлекательным
- Простая и понятная структура для удобства дальнейшего развития и поддержки
- Быстрый просмотр информации о услуге и преимуществах компании
